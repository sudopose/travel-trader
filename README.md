## How to Play

### Objective
Build your fortune by trading exotic goods across the world. Buy low, sell high, and unlock new destinations!

### Getting Started
1. **Start:** Begin in Istanbul or Mumbai with 200 gold coins.
2. **Trade:** Check the Market tab for current prices.
3. **Buy:** Purchase goods when prices are low (green indicators).
4. **Travel:** Move to new cities to find better selling prices.
5. **Sell:** Sell goods for profit!
6. **Unlock:** Save up to unlock luxury destinations like Beijing and New York.

### Tips for Success
- **Watch for Events:** Random market events can make prices skyrocket or crash.
- **Plan Routes:** Some cities are cheaper to travel to than others.
- **Consult Together:** One person can manage trades while the other navigates.

## Deployment

### Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import the repository: `sudopose/travel-trader`
3. Click **Deploy**

The `.vercel/project.json` file is already configured for Next.js deployment.

### Local Development
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
